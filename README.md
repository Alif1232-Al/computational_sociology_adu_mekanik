# computational_sociology_adu_mekanik

# 📊 Computational Sociology: Class Resentment & Digital Polarization
### *Analyzing the "Adu Mekanik" Phenomenon Among Indonesian Youth*

---

## 📌 1. Project Context & Overview
Proyek ini adalah sebuah **Computational Sociology Framework** yang dirancang untuk mensimulasikan dan menganalisis fenomena gesekan digital (populer dengan istilah **"Adu Mekanik"**) di kalangan generasi muda Indonesia di media sosial seperti TikTok dan X (Twitter).

Analis ini didasarkan pada metode **Netnographic Observation (Social Media Listening)** terhadap narasi konflik antara kelompok kerja lapangan (lulusan SMA/SMK) dengan kelompok akademisi (Anak Kuliahan). Proyek ini memetakan bagaimana tekanan makroekonomi—seperti tingginya angka pengangguran sarjana, stagnasi gaji lulusan baru di level UMR, serta glorifikasi semu jalur finansial berisiko tinggi (*Trader FOMO* / *Crypto* / *Forex* tanpa ilmu)—memicu kecemburuan kelas dan agresi verbal di ruang digital.

---

## 🧠 2. Core Hypotheses & Dataset Architecture
Karena keterbatasan dalam menangkap kondisi psikologis langsung dari komentar media sosial, proyek ini menggunakan pendekatan **Agent-Based Synthetic Population Framework (N=500 & N=1000 sampel)**. Variabel-variabel di dalamnya saling terikat menggunakan *Advanced Conditional Probability Loops* yang dikalibrasi berdasarkan realita kecemasan sosio-ekonomi pemuda Indonesia saat ini:

1. **Academic Disillusionment:** Model skeptisisme terhadap jalur kuliah akibat realita fenomena "Sarjana UMR".
2. **The Financial FOMO Loop:** Simulasi kelompok muda yang mengambil jalur pintas berisiko tinggi tanpa literasi keuangan dasar sehingga berakhir pada kegagalan finansial (loss trading/paylater).
3. **Class Resentment (Kecemburuan Sosial):** Bertindak sebagai katalisator psikologis utama yang didorong oleh ketimpangan gaya hidup dan kegagalan ekonomi.

---

## 🤖 3. Empirical Findings & Analytics Layer

Proyek ini dibagi menjadi beberapa tahapan analitik tingkat lanjut:

* **Machine Learning Predictive Modeling:** Menggunakan *Decision Tree Classifier*, model membuktikan bahwa **Tingkat Kecemburuan Sosial** memegang kendali tertinggi (~38%) sebagai prediktor utama konflik digital, disusul erat oleh faktor **Status Keuangan Akhir** (~24%).
* **Natural Language Processing (NLP):** Menggunakan ekstraksi fitur **TF-IDF** dan algoritma **Logistic Regression** pada teks komentar riil, AI berhasil mengidentifikasi struktur kalimat sarkasme dengan kata pemicu konflik utama seperti *'cuma'*, *'sarjana'*, dan *'jadi'*.
* **Correlation Matrix Analysis:** Ditemukan koefisien korelasi negatif yang kuat sebesar **-0.69** antara Status Ekonomi dengan *Online Toxicity Score*, membuktikan secara empiris bahwa tekanan ekonomi berbanding lurus dengan keagresifan verbal di internet.
* **Stochastic Monte Carlo Simulation:** Menggunakan fungsi probabilitas logistik non-linear (Sigmoid), model mengunci koordinat **Tipping Point (Threshold = 0.7)**. Ketika indeks tekanan ekonomi melewati angka 0.7, mekanisme pertahanan ego runtuh dan probabilitas netizen berubah menjadi *toxic* melesat secara eksponensial.
